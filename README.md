# Memory Game With React

## Usage

1. Clone the repo
```bash
 git clone https://github.com/orcunakkaya/Memory-Game.git
```

2. Go to file directory
```bash
 npm install
 npm run start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


## Built With

- React
- Redux Toolkit
- SASS
